=== ENVIRONMENT SETUP PLAN ===

1. DOWNLOADS NEEDED:
   - C++ Compiler (GCC 9+ / Clang 10+) with C++17 support (available in Ubuntu 20.04 repositories)
   - CMake 3.22 or higher (may need to add Kitware APT repository for Ubuntu 20.04 which ships with 3.16)
   - Meson 0.54.1 or higher (install via pip3)
   - Conan 1.63 (for Conan 1.x testing, install via pip3)
   - Conan 2.1 or higher (for Conan 2.x testing, install via pip3)
   - Conan Package Tools (cpt) - Python package for ConanMultiPackager
   - Bazel 7.0 or higher (download from GitHub releases for x86_64 Linux)
   - Bazelisk (download Linux x86_64 binary from GitHub)
   - Git (available in Ubuntu 20.04 repositories)
   - Python 3.8 (default in Ubuntu 20.04)
   - pip3 (python3-pip package)
   - codecov Python package (for code coverage reporting)
   - Catch2 v3.10.0 source code from GitHub repository
   - Ninja build system (ninja-build package)
   - clang-format (available in Ubuntu 20.04 repositories)
   - clang-tidy-15 or higher (may need LLVM APT repository)
   - Valgrind (available in Ubuntu 20.04 repositories)
   - libFuzzer (included with Clang)
   - AddressSanitizer (included with GCC/Clang)
   - Optional: lcov/gcov for code coverage (available in repositories)
   - Doxygen 1.9.1 or higher (may need to build from source or use newer APT repository)
   - Graphviz with dot tool (available in Ubuntu 20.04 repositories)
   - Optional: PlantUML for UML diagram support (requires Java)
   - Optional: MathJax for LaTeX formula rendering (download separately)
   - Optional: pkg-config (available in Ubuntu 20.04 repositories)
   - rules_cc 0.1.1 (automatically downloaded by Bazel with bzlmod)
   - bazel_skylib 1.7.1 (automatically downloaded by Bazel with bzlmod)
   - rules_license 1.0.0 (automatically downloaded by Bazel with bzlmod)
   - GitHub Actions cache action v3 (automatically available in GitHub Actions)
   - GitHub Actions checkout action v4 (automatically available in GitHub Actions)
   - vcpkg (optional, can be cloned from GitHub)
   - CTest (included with CMake)
   - MPI implementation (optional, openmpi-bin package)
   - JSON parsing capability in CMake (built-in with CMake 3.19+)
   - MarkdownSnippets .NET tool (requires .NET SDK installation)
   - .NET SDK 6.0 or higher (download from Microsoft for Linux x86_64)
